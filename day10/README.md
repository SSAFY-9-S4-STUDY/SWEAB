```python
for i in range(K - 1, N):
    for j in range(K - 1, M):
        ans.append(sr[i + 1][j + 1] - sr[i + 1][j - K + 1] - sr[i - K + 1][j + 1] + sr[i - K + 1][j - K + 1])
```
